# AI Chat Bot 🤖

An AI-powered full-stack chatbot application that provides intelligent, real-time conversations using modern web technologies. The project features a responsive user interface, secure backend APIs, user authentication, and AI integration for generating conversational responses.

---

## 🚀 Features

* 💬 Real-time AI chat interface
* 🔐 Secure user authentication
* 👤 Google OAuth login support
* ⚡ Fast and responsive UI
* 📱 Mobile-friendly design
* 📂 Chat history management
* 🔄 RESTful API architecture
* 🌙 Modern and clean interface
* 🛡️ Environment variable protection
* 📦 Modular project structure

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite

* JavaScript
* Axios

### Backend

* Node.js
* Express.js

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/NavanethaKrishnan-14/AI_Chat_bot.git
```

### Navigate to the project

```bash
cd AI_Chat_bot
```

### Install backend dependencies

```bash
cd chat-backend
npm install
```

### Install frontend dependencies

```bash
cd ../chat-frontend
npm install
```

---

## ▶️ Running the Project

### Start the backend

```bash
cd chat-backend
npm run dev
```

### Start the frontend

```bash
cd chat-frontend
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `chat-backend` directory.

Example:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```

**Do not commit your `.env` file to GitHub.**
