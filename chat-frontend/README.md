# AI Chat UI Package

Modular frontend chat widget built with React.js, Tailwind CSS, and Axios.

## Contents
- `src/components/ChatBot.jsx` - Widget frame and orchestration.
- `src/components/MessageBubble.jsx` - Message bubbles with Markdown syntax blocks.
- `src/components/TypingIndicator.jsx` - CSS animated loading states.
- `src/hooks/useChat.js` - Custom state hook with local persistence profiles.
- `src/services/chatbotApi.js` - Centralized downstream endpoints interface.
